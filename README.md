# Projeto Painel PROEP - CENET/FCEE

Este repositório contém o Painel de Monitoramento Georreferenciado do PROEP para a FCEE.

## Estrutura
- `/index.html`: Painel público com mapas e gráficos.
- `/admin_cenet.html`: Interface de administração para upload de dados.
- `/api/`: Lógica de processamento em nuvem (Vercel Python).
- `/data_proep.json`: Base de dados atualizada.

## Deploy na Vercel
O projeto está configurado para deploy automático via Vercel ao fazer push para este repositório.
